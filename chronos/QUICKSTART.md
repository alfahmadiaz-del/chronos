# Chronos - Quickstart Guide

This guide will get you started with Chronos in 30 seconds.

## Option 1: Use the Public Web App

1.  **Open the app:** Open your browser and go to [chronos.app/chronos.html](https://chronos.app/chronos.html)
2.  **Start tracking:** You can start creating chronicles and vows immediately.

## Option 2: Run Locally

1.  **Download:** Download the project files.
2.  **Open `chronos.html`:** Open the `chronos/chronos.app/chronos.html` file in your browser.

That's it! You are now using Chronos. All data is stored locally in your browser.
